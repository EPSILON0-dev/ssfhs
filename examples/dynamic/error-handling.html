This page explains how the SSFHS server handles errors with custom error pages.

  • 400 Bad Request: Occurs when requests are malformed.
  • 403 Forbidden: Access is denied to protected resources.
  • 404 Not Found: The requested resource does not exist.

Custom error pages offer a friendly and styled user experience with miku vibes.

All error pages currently serve static content.
